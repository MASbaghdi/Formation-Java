Introduction générale

L'informatique ?

L'informatique est le traitement automatique de l'information.

L'ordinateur ?

L'ordinateur est la machine qui permet de réaliser des opérations graçe à son système informatique.

Un système informatique est composé de deux parties :

1. Hardware :

Elle désigne le matériel physique (MC, CPU,...).

La mémoire centrale (MC) :
Central Processing Unit (CPU):

2. Software :

Elle désigne l'ensemble de logiciels et d'applications.

Un algorithme ?

Un algorithme est une suite finie et non ambiguë d’opérations ou d'instructions permettant de résoudre un problème.

La différence entre un algorithme et un programme ?

Un algorithme est écrit dans un langage naturel.

Un programme est écrit dans un langage de programmation compris par la machine.

L'exécution d'un algorithme doit toujours se terminer avec un résultat.

Exemple :
L’algorithme du café chaud

L'exécution d'un programme peut conduire à une boucle infinie (ne jamais s'arrêter).

Exemple :
Les systèmes d'exploitation .

La programmation ?

Le processus de transformation d'une solution théorique en une solution logicielle.

La variable

  • Occupe un espace dans la mémoire centrale.
  • Elle possède un nom et un type.
  • Elle a une valeur.
  • Doit être déclarée avant toute utilisation.
    nom_type nom_variable;
  • Le fait de lui donner une valeur est appelé affectation.
    nom_variable = valeur;
  • déclaration + affectation = initialisation. 
    nom_type nom_variable = valeur;

Règles sur le nommage de variables

  • Le nom doit commencer par a-z, A-Z, $ ou _.
  • Il peut contenir a-z, A-Z, 0-9, $ et _ mais pas d'espaces.
  • Il ne peut pas être un mot clé, un mot réservé ou une constante litérale :

Convention sur le nommage des variables

  • Le nom doit commencer par une minuscule (a-z).
  • Si le nom contient plusieurs mots le premier commencera par une minuscule et les autres par une majuscule.
    Exemple : 
    
byte groupe;
float moyenneEtudiant;
int nombreEtudiantsAbsents;

Les expressions arithmétiques

  • Une expression peut être la valeur d'une variable (entière ou réelle). 
    x, moyenneEtudiant, nombreEtudiantsAbsents, ...
  • Elle peut être une constante littérale. 
    0, 1, 13, -30, 0.2, ...
  • Le résultat d'une fonction. 
    f(x), Math.sin(expression), Math.pow(5, 2), ...
  • La combinaison d'autres expressions avec des opérateurs arithmétique. 
    20 * x + 2, 2 * Math.sin(x / 2), ...

Les expressions booléennes

  • Une expression peut être la valeur d'une variable (booléenne).
  • Elle peut être une constante littérale (true ou false).
  • Le résultat d'une fonction.
  • Le résultat d'une comparaison. 
    moyenneEtudiant < 10, x != y, nombreEtudiantsAbsents == 0
  • La combinaison d'autres expressions avec des opérateurs logiques. 
    !((moyenneEtudiant >= 0) && (moyenneEtudiant <= 20)), ...

Les structures conditionnelles

Permettent de réaliser des actions suivant qu'une certaine condition est satisfaite ou non.

Le if

La syntaxe :


if (expression_booléenne) {
	instruction0;
	instruction1;
	...
}

Exemple :


if (nombreEtudiantsAbsents == nombreEtudiants) {
	System.out.println("Tout le monde est absent.");
}

Le if else

La syntaxe :


if (expression_booléenne) {
	...
} else {

}

Exemple :


if (moyenneEtudiant >= 10) {
	System.out.println("Admis");
} else {
	System.out.println("Ajourné");
}

Le if else if ... else

La syntaxe :


if (expression_booléenne) {
	...
}
else if (expression_booléenne) {
	...
}
else if (expression_booléenne) {
	...
}
...
else {
	...
}

Exemple :


if (moyenneEtudiant < 8) {
	System.out.println("Faible");
} else if (moyenneEtudiant >= 8 && moyenneEtudiant < 12 ) {
	System.out.println("Moyen");
} else if (moyenneEtudiant >= 12 && moyenneEtudiant < 17) {
	System.out.println("Bon");

Les structures répétitives

Ils permettent de répéter un ensemble d'actions un nombre déterminé de fois, ou tant qu'une condition est satisfaite.

La boucle while

La syntaxe :


while (expression_booléenne) {
	instruction0;
	instruction1;
	...
}

Exemple :


int i = 0;
while (i < 10) {
	System.out.println("Hello World !!!");
	i = i + 1;
}

La boucle for

La syntaxe :


for (initialisation; condition; mise_a_jour) {
	instruction0;
	instruction1;
	...
}

Exemple :


for (int i = 0; i < 10; i = i + 1) {
	System.out.println("Hello World !!!");
}

La boucle do while

La syntaxe :


do {
	instruction0;
	instruction1;
	...
} while (expression_booléenne);

Exemple :


int i = 0;
do {
	System.out.println("Hello World !!!");
	i = i + 1;
} while (i < 10);

Les tableaux

Un tableau est une collection statique et ordonnée de variables.

  • Déclaration d'un tableau : 
    
nom_type[] nom_tableau;
  • Création d'un nouveau tableau : 
    
nom_tableau = new nom_type[taille_tableau];
  • Accéder à un élément avec son indice (le premier élément est à l'indice 0) : 
    
nom_tableau[indice] = valeur;
  • Récupérer le nombre d'éléments : 
    
nom_tableau.length
